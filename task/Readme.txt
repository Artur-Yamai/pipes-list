Создать простой SPA, используя Vue.js

Приложение должно состоять из главной страницы и страницы справочников.

На главной странице сделать таблицу трубопроводов
Трубопроводы - массив, элементы которого содержат следующие свойства:
id - идентификатор
name - название трубопровода
purposeId - идентификатор назначения трубопровода
stateId - идентификатор состояния трубопровода
diameter - диаметр трубы, мм
thickness - толщина стенки трубы, мм
formationDate - дата установки трубы

purposeId и stateId - идентификаторы из справочников purposes (Назначения) и states (Состояния), каждый из которых состоит из свойств id и value (Значение).

Доступные операции для списка трубопроводов:
- Добавить новую трубу
- Редактировать выбранную трубу
- Удалить выбранную трубу

Для добавления и редактирования должно использоваться диалоговое окно.
При подтверждении ввода данных необходимо проводить валидацию, все поля должны быть заполнены. Подумать какая еще возможно нужна валидация.

На странице справочников сделать вывод списка справочников (2 справочника).
При выборе определенного справочника, выводить список значений соответствующего справочника.
Операции добавления, редактирования и удаления в справочниках реализовывать не нужно.

Допускается использование любых сторонних пакетов, в том числе и UI-фреймворков (Bootstrap, Vuetify, Quasar и пр.)
Для эмуляции REST API можно использовать json-server
